Для создания авто теста на проверку статус-кода страницы проделываем следующие шаги:
1. Создаем новый запрос, называем его "Проверка статус-кода" или "Проверка отправки данных"
2. Выбираем метод POST
3. В строку url вставляем адрес нашей страницы
4. Переходим во вкладку Test
5. Cлева выбираем сниппет: "Status code: Code is 200"
6. Нажимаем на Send
7. Если наш код будет со статусом 200 - в консоли отобразится PASS, если нет, то - FAIL.
